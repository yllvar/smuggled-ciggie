# Blog Visualizations

This directory contains all the visualizations extracted from the Illicit Cigarettes Study report and generated by our analysis scripts. These images can be used to populate the blog post.

## Generated Visualizations

### Market Analysis
- `market_analysis_visualization.png` - Market share and tax loss overview
- `state_tax_loss_chart.png` - Geographic impact visualization
- `enforcement_roi_analysis.png` - Strategy comparison and optimization

### Interactive Dashboards
- `interactive_brand_treemap.html` - Brand market share treemap
- `interactive_incidence_map.html` - State-level incidence mapping
- `interactive_roi_dashboard.html` - Enforcement ROI calculator
- `interactive_timeseries.html` - Time series forecasting explorer

## Extracted Report Visualizations

The following images were extracted directly from the original PDF report:
- `report_*.png` - Various charts and graphs from the original report
- `report_*.jpg` - Various images and figures from the original report

## Usage

To use these visualizations in your blog:

1. For PNG/JPG images: Simply include them in your blog post with appropriate alt text and captions
2. For HTML files: These are interactive dashboards that can be embedded in your blog or hosted separately

## Notes

- Some extracted images may be duplicates or low quality
- Review all images to select the most relevant ones for your blog
- The interactive HTML files require web hosting to function properly
